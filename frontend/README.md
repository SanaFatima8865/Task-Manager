# Task Client
A React frontend for the Task Manager app, with JWT-authenticated task management.

## Live URL
https://task-manager-sana-fatima.vercel.app

## Features
- User registration and login with JWT auth
- Protected routes with route guarding
- Full CRUD for tasks via the Task API
- Axios-based API client with configurable backend URL

## Tech Stack
React · React Router · Axios

## Local Setup
1. Clone repo
2. npm install
3. Create .env with REACT_APP_API_URL (defaults to http://localhost:5001)
4. npm start
